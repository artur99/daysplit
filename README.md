### Instalare
Cerințe: `git`, `composer`

```sh
$ git clone https://github.com/artur99/daysplit.git
$ composer install
$ mysql -u root -p < db_structure.sql
```

### Dezvoltare

Cerințe: `git`, `composer`, `node.js`

```sh
$ npm install -g bower
$ npm install -g grunt-cli
$ git init
$ git pull https://github.com/artur99/daysplit.git
$ npm update
$ bower update
$ grunt
$ composer install
$ mysql -u root -p < db_structure.sql
```
